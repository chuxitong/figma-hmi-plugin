# Неделя 2. Индустриальный набор экран-макапов как PNG

По задаче недели нужен небольшой, но представительский набор HMI-досок для всех следующих недель экспериментов.

## Что сделано

- Исходник макета лежит в `mockups/layered-svg/screens/*.svg`; растр **1440×900** воспроизводим командой из `mockups/build_mockups.py` (Playwright) в `mockups/png/*.png`. Так эталоны не зависят от случайной ручной нарезки в Figma.
- В экспериментальном пайплайне Week 8 по-прежнему используются эти восемь именований экранов; для сравнения «реальный экспорт переменных» в ВКР — отдельный Week 7 `from-json` с `reports/deliverables_week7/figma_native/`.

## Где артефакты

| Путь | Комментарий |
|------|-------------|
| `mockups/png/*.png` | Финальный набор экспериментальных эталонов |
| `mockups/layered-svg/screens/` | Авторские SVG откуда воспроизводится PNG |

## Замечание для отчётности Давыдову

PNG из SVG — всё равно **графический эталон**; они не должны смешиваться с выводами о **переменных из Figma** — для этого используется плагин + `payload.json`.
