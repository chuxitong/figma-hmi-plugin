# Неделя 5. Рендер HTML→PNG и первый интерфейс плагина

По недельному плану задача включала два блока:

1. **Backend:** Playwright-/Chromium-слой для отрисовки HTML в PNG (используется и локальным пайплайном оценки, и кнопкой Refine после `/render`).
2. **Плагин:** панель с Generate / closer-to-mockup / Apply edit, лог, поле базового URL.

## Связка с финальной выгрузкой

Демонстрационные материалы остаются в `figma-plugin/src/ui.html` + `dist/code.js`; сборка `npx tsc` нужна только после правок sandbox. Современные исправления (буфер обмена, сбор boundVariables из Libraries) перечислены в `help/必须同步到本机的文件清单.md`.

## В контексте ВКР

Неделя 5 из **оригинального** плана (рендер и оболочка) **не** относится к «скриптовому синтетическому ablation-блоку» Week 7/`synthetic`; последний переименован в инженерный приём без обязательного включения в тезис.
